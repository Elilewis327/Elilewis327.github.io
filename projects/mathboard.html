Mathboard

Posted on 12/28/2021

The Purpose

I created Mathboard for my final Cs 108 project, as a way to help me do my math homework. I wanted something that I could write equations out onto, and also have easy acess to a graphing calculator.

Creating It

We had been learning the python binding's for tkinter throughout that semester, so I used the tkinter canvas to create the whiteboard section.
I implemented:
  • The whiteboard
  • Different drawing colors
  • A textbox
  • A bad graphing calculator
  • Some other simple features
Looking a year back on when I did it now, I would do a lot different today, such as
  • Organize the code more by seperating the state and the gui better
  • create a much better equation parser for the graphing calculator
  • not use python tkinter bindings (I think QT or GTK would be much nicer to use)
